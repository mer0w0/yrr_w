# ゆるるーわー（yrr_w）

「ゆるるーわー」は、ゆる〜くURLを表示する自己利用向けの軽量ビューアです。  
Render や GitHub Codespaces で簡単に動かせます！


    
## 🌐 サイト概要

URLを入力して「Go」ボタンを押すと、対象ページを新しいタブで開きます。  


## 🚀 デモ（Render）

（※初回起動には数秒かかります）

## 🔧 使い方

### 1. コードスペースで起動する場合（推奨）

1. GitHubでこのリポジトリをForkまたはClone
2. 「Code」 → 「Codespaces」 → 「New codespace」 をクリック
3. ターミナルで以下を実行：

```bash
npm install
npm start
